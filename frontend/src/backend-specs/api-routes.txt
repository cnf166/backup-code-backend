BASE_URL: http://localhost:8000/api/v1

--- ROUTER REGISTRY (routes/v1/__init__.py) ---
Prefix structure: /api/v1 + router_prefix
Active Routers:
- /orders (Booking)
- /orders/items (Booking Items)
- /payments (Payments)
- /resources/dishes
- /resources/tables
- /resources/equipments
- /resources/ingredients
- /feedbacks

--- DETAIL ENDPOINTS ---

1. ORDERS (routes/v1/booking/Order.py)
GET    /orders              -> List[OrderRead] (Filter: table_id, status_id, guest_id)
POST   /orders              -> OrderRead (Body: OrderCreate)
GET    /orders/{id}         -> OrderRead
PUT    /orders/{id}         -> OrderRead (Body: OrderUpdate)
DELETE /orders/{id}         -> OrderRead
GET    /orders/{id}/total   -> Decimal (Calculate total amount)

2. ORDER ITEMS (routes/v1/booking/OrderItem.py)
GET    /orders/items        -> List[OrderItemRead]
POST   /orders/items        -> OrderItemBase (Body: OrderItemCreate)
PUT    /orders/items/{id}   -> OrderItemBase
DELETE /orders/items/{id}   -> OrderItemBase

3. PAYMENTS (routes/v1/payments/Payment.py)
GET    /payments            -> List[Payment]
POST   /payments            -> Payment (Body: PaymentCreate)
GET    /payments/{id}       -> Payment
PUT    /payments/{id}       -> Payment (Body: PaymentUpdate - update status)
POST   /payments/{id}/refund

4. RESOURCES
GET    /resources/dishes    -> List[DishRead]
GET    /resources/tables    -> List[TableRead]
GET    /resources/ingredients -> List[IngredientRead]
